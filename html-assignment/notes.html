5 HTML tags that I have learned

img src="./images/.jpg" alt="image"
The HTML img tag is used to embed an image in a web page. Images are not technically inserted into a web page, images are linked to web pages. The img tag creates a holding space for the referenced image.

table
I have learned how to input a table, we can use it to make information and elements look more oraganized and in unison.

a href="Link" target="_blank"> description
I have learned how to add an external link to my website, using the target will allow the viewer of the website to click the link, whilst staying on the main website.

& dollar;
Using an ampersand, dollar and a semi colon, we are able to create a dollar sign. $

blockquote and cite
Using a blockquote and a cite, we are able to properly cite a quote.
